# StratAtlas v0.1.11

Date: 2026-03-09
Status: Planned release entry for `WP-GOV-INSTALLER-003`

## Highlights

- Governed release checks now validate the live current installer surface, portable executable surface, timestamped installer kit, and migrated legacy archive state together.
- The maintenance menu assertion now reads the actual current changelog copy from the installer surface instead of pinning a stale version string.
- Installer and executable outputs remain excluded from git tracking across current, archived, and old-version release folders.
- Legacy `.product/build_target/Current` history is now migrated into a versioned `LegacyCurrent` archive folder and drained from the legacy root.

## Operator Notes

- The current release remains under `.product/build_target/Releases/Current/`.
- Prior governed releases and the migrated legacy installer history remain under `.product/build_target/Releases/Archive/<version>/`.
- Default uninstall and repair flows preserve user data and presets unless a full clean path is explicitly chosen.
