# StratAtlas v0.1.7

Date: 2026-03-09
Status: Planned release entry for `WP-GOV-INSTALLER-003`

## Highlights

- Governed release outputs now publish through `.product/build_target/Releases/Current/` and archive prior builds by version under `.product/build_target/Releases/Archive/`.
- Installer, setup EXE, MSI, and portable EXE artifacts remain gitignored across current and archived release folders.
- The Windows maintenance script now exposes a plain-language menu with install, uninstall, full uninstall, repair, full repair, update, downgrade, status, and changelog actions.
- Installer kits now bundle lifecycle guidance plus the governed changelog entry as `CHANGELOG_CURRENT.md`.

## Operator Notes

- Standard uninstall preserves user presets and user data unless `full-uninstall` is explicitly chosen.
- `full-repair` performs a clean reinstall while restoring backed-up user data by default; use `-DropUserData` only for a fully clean reinstall.
- Historical release artifacts remain separated by version so prior installer generations are not mixed into the current release surface.
