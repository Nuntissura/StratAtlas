# StratAtlas v0.1.6

Date: 2026-03-09
Status: Planned release entry for `WP-GOV-INSTALLER-003`

## Highlights

- Release artifacts move to the governed `Releases/Current` and versioned `Releases/Archive` layout.
- Installer, setup EXE, and portable EXE outputs stay gitignored under build-target release folders.
- Windows maintenance tooling now exposes a plain-language menu/help surface with changelog output.
- Maintenance options now distinguish:
  - uninstall while preserving user presets/data
  - full-uninstall that also removes user presets/data
  - repair
  - full-repair / clean reinstall with restore-by-default behavior

## Operator Notes

- Use the governed maintenance menu to review lifecycle options before running repair or uninstall flows.
- Historical release artifacts are archived separately by shipped version; the latest release remains under `.product/build_target/Releases/Current/`.
