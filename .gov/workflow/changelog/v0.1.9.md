# StratAtlas v0.1.9

Date: 2026-03-09
Status: Planned release entry for `WP-GOV-INSTALLER-003`

## Highlights

- Governed Windows release promotion now passes through a versioned archive, current installer surface, portable executable surface, and timestamped installer kit in one auditable flow.
- Release kits keep the matching governed changelog and lifecycle guidance alongside installer assets for operator review.
- Installer and executable outputs remain gitignored across current and archived release folders so shipped binaries never pollute tracked repo state.
- Maintenance menu coverage remains explicit for install, repair, uninstall with data preserved, full uninstall, and full clean reinstall behavior.

## Operator Notes

- Use the maintenance menu before repair or uninstall to review data-handling consequences.
- Standard uninstall preserves user data and presets; full uninstall removes them.
- Full repair performs a clean reinstall while restoring backed-up user data and presets unless a clean wipe is explicitly requested.
