# StratAtlas v0.1.8

Date: 2026-03-09
Status: Planned release entry for `WP-GOV-INSTALLER-003`

## Highlights

- Governed installer promotion now validates the `Releases/Current` plus `Releases/Archive/<version>/` topology end to end.
- Release kits continue to copy lifecycle guidance and the governed changelog into the current installer surface as operator-facing documentation.
- Installer and portable executable outputs remain gitignored across current, archived, and old-version build-target folders.
- Maintenance menu coverage remains explicit for install, repair, uninstall-preserving-data, full uninstall, and full clean reinstall flows.

## Operator Notes

- Standard uninstall keeps user data and presets unless a full uninstall path is deliberately chosen.
- Full repair performs a clean reinstall with restore-by-default behavior for backed-up user data and presets.
- Archived governed releases stay separated by shipped version so older installer generations remain recoverable without contaminating the current release surface.
